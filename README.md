# EDP_liecrew - AtCoder Educational DP Contest 回答リポジトリ

このリポジトリは、競技プログラミングサイト[AtCoder](https://atcoder.jp/home) の「[Educational DP Contest](https://atcoder.jp/contests/dp?lang=ja)」における自分の回答を整理・管理するためのものです。

- 各問題ごとにPythonファイルを保存しています。
- コードの見直しや復習、精度向上のために継続的に更新していきます。

## ファイル構成

```
EDP_liecrew/
├── myans_py/                # 解答ファイル格納フォルダ
├── main.py                 # 解答ファイル
├── main2.py                # 解答ファイル
├── requirements.txt
└── README.md
```
